VisModel.js
====

Fast tree drawing library

How to build
----
```
git clone https://github.com/tetsurom/vmjs
cd vmjs
npm install
gulp
```

Dependencies
----
* [polymer-gestures](https://github.com/Polymer/polymer-gestures)
